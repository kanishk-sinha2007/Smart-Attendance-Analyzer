# Smart Attendance Analyzer

## 📌 Overview

Smart Attendance Analyzer is a Python-based tool that helps students track and manage their attendance effectively. It provides insights into attendance percentage, risk level, and future requirements to maintain minimum attendance.

---

## ❗ Problem

Students often fail to track attendance properly and realize shortages too late. This project helps solve that by giving real-time analysis and predictions.

---

## 💡 Features

* Calculate attendance percentage
* Identify risk (Safe / At Risk)
* Predict classes needed to reach minimum attendance
* Suggest how many classes can be skipped
* Simple and easy-to-use interface

---

## 🛠️ Tech Stack

* Python
* (Optional) Streamlit / Flask
* Matplotlib (for graphs)

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/smart-attendance-analyzer.git
```

2. Navigate to the project folder:

```bash
cd smart-attendance-analyzer
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the program:

```bash
python main.py
```

Enter:

* Total classes
* Attended classes
* Required percentage

The system will display:

* Current attendance
* Status
* Recommendation

---

## 📊 Example

Input:

* Total Classes: 50
* Attended: 30
* Required: 75%

Output:

* Attendance: 60%
* Status: At Risk
* Suggestion: Attend next X classes
